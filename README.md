### **Playtech - reports**

This component will generate a CSV in project root folder as per below conditions
 1.	A CSV file named session_wagered_won.csv that shows each player’s total wagered and won amount per session.
 2.	A CSV file named game_wagered_won.csv that shows each player’s total wagered and won amount per game.
 3.	A CSV file named session_game_wagered_won.csv that shows each player’s total wagered and won amount per game per session.
 4.	A CSV file that contains IDs of 5 players that have been most profitable for the company.

**App**

    Java: 11 

**Run locally**

To run application locally: Run `PlayTechApplication.java` file which has main method.

**Test**

To run tests(Unit and Integration test), use:
    `./gradlew clean test`

**IDE Used**
   `IntelliJ IDEA`
